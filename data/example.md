## Getting Started

To get started with this project:

1. Clone the repository
2. Install dependencies with `npm install`
3. Start the server with `npm start`

## Features

- **Fast Search**: Uses ripgrep for instant results
- **Mobile Optimized**: Works great on iOS Safari
- **Markdown Rendering**: Beautifully displays markdown content
